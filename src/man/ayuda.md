```bash
# Título
.TH # Despúes de TH ponemos el titulo, si pones un espacio representará entre parentesis una indicacion acerca del titulo

# Subtitulo
.SH # Despúes de .SH ponemos un texto indicando el subtitulo

# Resaltar texto
.B # Resaltamos texto con solamente poner el texto despúes de .B

# Cambio de parrafo
.PP # Doble salto de linea o también conocido como "Salto de parrafo". Sin texto a continuacion.

# Salto de línea
.br # Simple salto de linea. Sin texto a continuacion.

# Subrayar texto
.IR # Subraya el texto a continuacion

# No se que mierd*s haga
.TP # No se que mierd*s haga

# Ejemplo
.BR \-n ", " \-\-bits =\fIBITS\fR

<<lat

Vease más en:
http://liw.fi/manpages/

lat
```
